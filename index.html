enlightened by vincentmac

A dark color scheme for Sublime Text 2

Install

Download the files using the GitHub .zip download option Unzip the files Copy the .tmTheme to your Packages/Color Scheme - Default folder. Choose enlightened from Preferences > Color Scheme

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/vincentmac/enlightened
